# ProviSys

### Descripción del Proyecto

**ProviSys** es una solución digital creada por NovaTechZeo con el objetivo de ayudar a **expandir negocios en el sector de la hostelería**.

Su propósito es facilitar y optimizar las herramientas necesarias para lograr una **mejor comunicación** y gestión eficiente entre las empresas de hostelería y sus proveedores.

---

## Quienes somos los desarrolladores del grupo

Este proyecto ha sido realizado por **NovaTechZeo**, una empresa de Desarrollo de Software especializada en ofrecer soluciones digitales a problemas reales en negocios de diversas orientaciones.

---

## Partes de la Aplicación

ProviSys está construido con las siguientes tecnologías:

| Componente | Tecnología | Enfoque Principal |
| :--- | :--- | :--- |
| **Frontend** | **JavaScript** & **Electron** | Desarrollo de una aplicación de **escritorio** multiplataforma. |
| **Diseño UI** | **Material-UI (MUI)** | Componentes para una interfaz de usuario moderna y profesional. |
| **Backend** | **Node.js** & **Express.js** | Servidor API RESTful para la lógica de negocio. |
| **Base de Datos** | **MongoDB** | Almacenamiento de datos NoSQL flexible y escalable. |

---

## Como ponerlo en marcha

Para iniciar el proyecto localmente:

1.  Clona los repositorios de Frontend y Backend.
2.  Instala dependencias en cada carpeta: `npm install`.
3.  Asegúrate de que tu conexión a **MongoDB** esté configurada en el servidor de Express.
4.  Inicia la API (Backend): `node server.js` (escuchando en `http://localhost:3000`).
5.  Inicia el cliente (Frontend/Electron): `npm start`.
